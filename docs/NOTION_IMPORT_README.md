# How to use these docs in Notion

These Markdown files are written so you can create a **Notion workspace** for the CobbleDollars Villagers Overhaul
project.

## Option 1: Copy-paste

1. In Notion, create a **parent page** (e.g. "CobbleDollars Villagers Overhaul – Docs").
2. For each file below, create a **sub-page** under it and paste the contents of the corresponding `.md` file. Notion
   supports Markdown; headings, lists, and code blocks will format correctly.

## Option 2: Import

1. In Notion, go to **Import** (e.g. from Settings or the sidebar).
2. Choose **Markdown & CSV** and select the `.md` files. Each file can become one page.

## Suggested Notion structure

| Notion page                 | Use this file                      |
|-----------------------------|------------------------------------|
| **Project overview** (root) | `01_Project_Overview.md`           |
| **Common issues**           | `02_Common_Issues.md`              |
| **Configuration**           | `03_Configuration.md`              |
| **Trading behavior**        | `04_Trading_Behavior.md`           |
| **Discord & support**       | `05_Discord_Support_Guidelines.md` |

---

**Files in this folder**

- `01_Project_Overview.md` – Overview, features, requirements, installation
- `02_Common_Issues.md` – Common issues and fixes (shop not opening, wrong price, item not received, etc.)
- `03_Configuration.md` – Config options, default shop/bank, edit screen
- `04_Trading_Behavior.md` – How buys work (costA, costB, CobbleDollars), for debugging/support
- `05_Discord_Support_Guidelines.md` – What to ask users when they report issues (loader, version, template)
