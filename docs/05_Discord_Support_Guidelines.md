# Discord & support guidelines

What to ask when someone reports a bug or needs help.

---

## Before they post

- **Search** – Check pins and recent messages.
- **Right channel** – Use support/bug channels, not general chat.

---

## What to ask for (every report)

Have them include:

| Field                  | Example                                                     |
|------------------------|-------------------------------------------------------------|
| **Loader**             | Fabric **or** NeoForge                                      |
| **Loader version**     | e.g. Fabric 0.17.x, NeoForge 21.1.x                         |
| **Minecraft version**  | e.g. 1.21.1                                                 |
| **Mod version**        | e.g. 0.1.1.5-hotfix (from mod list or JAR name)             |
| **What happened**      | Short description                                           |
| **What they expected** | Short description                                           |
| **Other mods**         | Cobblemon, CobbleDollars, RCT versions if relevant          |
| **Screenshot / log**   | For GUI issues or crashes, latest log snippet or screenshot |

Without **Fabric vs NeoForge**, **Minecraft version**, and **mod version**, it’s hard to tell if the issue is already
fixed or loader-specific.

---

## Copy-paste template (for users)

```
Loader: Fabric / NeoForge
Loader version:
Minecraft version:
Mod version (CobbleDollars Villagers Overhaul):
What happened:
What I expected:
Other mods (Cobblemon, CobbleDollars, etc.):
Screenshot / log: (attach or link)
```

---

## Behavior rules

- Be respectful; no spam or unprompted self-promo.
- Use the right channel for bugs, suggestions, and off-topic.
- Spoiler long pastes or sensitive content if the server asks for it.

---

## What we can’t help with

- **Pirated / unsupported clients** – Only official Minecraft and standard loaders.
- **Huge modpacks** – Suggest testing with only CobbleDollars + Cobblemon + this mod.
- **Manual config edits** – We can give guidance; users should back up configs first.
